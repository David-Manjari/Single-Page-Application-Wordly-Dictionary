# Single Page Application - Wordly Dictionary

## Description
This is a project  curated to help user with looking up words in a dictionary. It provides a summary of the word entered by the user which is inclusive of:
 - The word
 - A sample of pronunciation
 - Word definition
 - Sample audio of pronunciation
 - Examples
 - The type of word it is (Noun, Verb)
 
 ## Features
 - Search Word Definitions
 - Display Pronunciation
 - Audio playback
 - Display **Synonyms** and **Antonyms**
 - Error Handling for **_Invalid Words_** and **_Empty Input_**

 ## How to Run 
1. Visit my GitHub repository through the provided link
2. Download or clone the project
3. Open `index.html` in your browser
4. Enter a word in the search bar and click search

## Technologies Used
- **HTML5** – Structure of the application  
- **CSS3** – Styling and layout  
- **JavaScript** – Application logic and DOM manipulation  
- **Free Dictionary API** – Fetching word meanings and pronunciation data 

##  Known Issues
- Some words may not have audio
- Some words have double audio
